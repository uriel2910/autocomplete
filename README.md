# React TypeScript Autocomplete Component

This project demonstrates how to create a simple autocomplete component in React using TypeScript, without any third-party libraries. It uses mock data and simulates an API call using `setTimeout` to mimic async behavior. The matching part of the text is highlighted in the results.

## Features
- Async filtering of results
- Keyboard navigation using arrow keys
- Highlighting of matched parts of text
- Basic styling and UX
- Mock data simulating an API response

## Setup

### Prerequisites
You will need Node.js and npm installed on your system.

### Running the Project

1. Clone the repository:
https://github.com/uriel2910/autocomplete.git

2. Navigate to the project directory

3. Install dependencies:
npm install or yarn install

4. Run the project:
npm start or yarn start

The project will be served at `http://localhost:3000`.
